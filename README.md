## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- Yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/smondal1174/Subhadeep_Mondal_Portfolio.git

# Navigate to frontend directory
cd Subhadeep_Mondal_Portfolio/frontend

# Install dependencies
yarn install

# Start development server
yarn start
```

The application will open at `http://localhost:3000`

### Build for Production

```bash
yarn build
```
